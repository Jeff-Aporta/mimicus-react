# Isla display

Componentes de presentación de datos (Data Display): tablas, listas, etiquetas, avatares, líneas de tiempo, estadísticas, tooltips, etc. Para mostrar información al usuario.

## Exporta
- Etiquetas/marcas: `Badge`, `Tag`, `Chip`, `Segmented`
- Avatares: `Avatar`, `AvatarGroup`
- Contenedores expandibles: `Carousel`, `Collapse`, `CollapsePanel`, `Accordion`
- Descripciones: `Descriptions`, `DescriptionsItem`
- Estado/medidas: `Empty`, `QRCode`, `Statistic`
- Datos tabulares/jerárquicos: `Table`, `Tree`, `Calendar`, `Timeline`, `TimelineItem`
- Listas: `List`, `ListItem`, `ListItemText`, `ListItemIcon`, `ListItemAvatar`
- Ayuda contextual: `Tooltip`, `Tour`
- Boot: `mountDisplay`

## Depende de
Isla `theme`.

## Uso
import { Table, Tag, Tooltip } from "@jeff-aporta/mimicus-react/display";
// + cargar el CSS: @jeff-aporta/mimicus-react/display.css
