# Isla navigation

Componentes de navegación: menús, tabs, breadcrumbs, paginación, pasos, drawers, enlaces y acciones rápidas. Para moverse entre vistas y secciones.

## Exporta
- Enlaces/anclas: `Anchor`, `NavLink`, `Link`
- Migas: `Breadcrumbs`, `Breadcrumb`
- Menús: `Dropdown`, `Menu`
- Pestañas: `Tabs`, `TabItem`
- Paginación: `Pagination`
- Pasos: `Steps`, `Step`, `Stepper`
- Paneles/barras: `Drawer`, `BottomNavigation`, `BottomNavigationAction`
- Acciones rápidas: `SpeedDial`, `SpeedDialAction`
- Boot: `mountNavigation`

## Depende de
Isla `theme`.

## Uso
import { Menu, Tabs, Breadcrumbs } from "@jeff-aporta/mimicus-react/navigation";
